Formatação de Strings

Existem comandos que alteram o tipo dos elementos do código, podendo transformar strings em números, número em boolean, etc.. Cada comando exerce uma função diferente na formatação dos Strings, um não faz interpolação, outros fazem concatenação, além disso, ainda existe teplate de String.

Oque é interpolar e concatenar?

Interpolar é a junção de elementos bruta, nela os elementos ficam juntos, mas sem somar os elementos, apenas justapondo eles. Como exemplo, se hipoteticamente fizermos uma calculadora digital e colocarmos a string no formato
