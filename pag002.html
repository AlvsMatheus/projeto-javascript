Data Types

Data Types: Traduzindo literalmente para o português seria algo como "tipos de dados", referindo-se aos dados que guardamos nas variáveis. É importante conhecer pelo menos os tipos mais usados de dados mais, pois há diferença no modo de uso de cada um deles. Abaixo está uma lista de alguns Data Types comuns no JavaScript.

NUMBER:

São os dados do tipo número inteiros e reais {0, 1, -2, 3.14, 18}. Existem muitos outros tipos de dados nos referirmos a números, como Float - que se refere aos números que apresentam o ponto flutante ".", ou seja, números "quebrados" (Reais), como o número pi (3,14), contudo, também há os Integers - que se referem aos números inteiros, como 1, 6, 10, -1, -30 etc...

STRING:

São os dados do tipo "conjunto de caracteres", para adicionar ele no JavaScript é necessário adicionarmos aspas, aspas duplas ou crase ("", ` ` ou '') veremos o significado de cada uma no resumo Strings. Basicamente, strings podem ser nomes, endereços, e até números que contenham sinais específicos.
Ex: "Maria", "Brasil", "123.456.789-00" (É string pois possui o sinal "-")

BOOLEAN:

São os dados de afirmação (true or false), que usamos para validar a informação de um dado ou não. Como por exemplo, se eu dou o número 5 a uma variável(chamada de x), e o número 2 a outra variável (chamada de y) e depois escrevo o comando x == y (X é igual a y - o sinal de igual em JS é simbolizado por 2 iguais), o resultado será False, porque não apresenta uma igualdade verdadeira
