Operadores Gerais

Os operadores são os simbolos que utilizamos para retratar operações matemáticas nos códigos, ou para manipular um valor boolean.

Operadores Aritméticos

Os operadores padrões são os que utilizamos em cálculos normais, como em uma calculadora. São eles:

  • Adição ( + ): (5 + 5 = 10)
  • Subtração ( - ): (8 - 4 = 4)
  • Multiplicação ( * ): (8 * 4 = 32)
  • Divisão ( / ): (8 / 4 = 2)
  • Resto da divisão ( % ): (5 % 2 = 1)
  • exponenciação ( ** ): (4**2 = 16)

Operadores Relacionais

São os operadores utilizamos para estabelecer comparação entre dois valores

  • == : Idêntico; não em significado ( 5 == 5 True)
  • ! : Not ( Torna operadores de "aprovação" em negação )
  • != : não igual (5 != 2 True)
  • > : Maior que (5 > 2 True)
  • < : Menor que (20 < 40 True)
  • >= : Maior ou igual que (2 >= 2 True)
  • <= : Menor ou igual que (4 <= 3 True)

Operadores de atribuição

Simbolos utilizados para atribuir valores às variáveis

  • = : recebe (let x = 5)
  • += : adicionar e receber (x += 2, logo, x = x+2)
  • -= : subtrair e receber (x -= 1, logo, x = x-1)
  • *= : multiplicar e recceber (x *= 2, logo, x = x*2)
  • /= : dividir e receber (x /= 4, logo, x = x/4)

Operadores Boolean

Operadores boolean trabalham para manipular os valores dos dados boolean(True or False)

Operadores Lógicos

Usados para combinar ou modificar os Boolean expressions

  • AND : é True caso ambas afirmações forem True.
    ex: (5 > 2 and 4 < 6)
  • OR : é True caso a outra condição seja True
    ex: (5 > 2 or) or (4 < 6)
  • Not : Nega um valor Boolean
    ex: not (5 > 2), logo, False, pois 5 > 2 é True

Hierarquia de operadores

Os operadores em JS, assim como na matemática, precisam obedecer uma hierarquia para que o cálculo seja correto.

  • Parênteses
  • Exponenciação
  • Multiplicação, divisão e módulos (*, /, %)
  • Adição e subtração (+ e -)
  • Comparações (==, !=, >, <, >= <=)
  • Not (not)
  • And (and)
  • Or (or)
